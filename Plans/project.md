## Compact Finite Difference

### Motivation

We are developing new compact finite difference operators for numerical relativity. These operators will be used on distributed patches with MPI. Recent operators by Kim et al. have very good boundary closure properties.  the most recent paper on this method is @/Users/dave/Research/compact_finite_difference/Kim/kim-stable-boundary-closure.pdf.  I will call this paper Kim24.

We have extended Kim's method to generate classes new operators, for both first and second derivatives.  We are need to derive new operators for second derivatives that are sixth-order and have better accuracy and stability.

### Background

The key parts of Kim's method are described in a couple of papers.  Review these papers as part of this setup.

1. The paper @/Users/dave/Research/compact_finite_difference/Kim/Kim-optimization.pdf (call this Kim07) presents a method for constructing compact finite difference stencils for grid interiors using a error optimization algorithm.  An error function minimizes the spectral error with respect the free parameters in the stencil.
2. In general, the minimization process can include the boundaries.  The Kim07 paper does include a boundary closure, but it is inferior to the later Kim24.
3. The paper Kim24 provides a more accurate mroe stable boundary closure for the finite derivative operators based on an extrapolating polynomial.
4. The Kim07 and Kim24 papers only construct first derivative operators.  We want to extend this approach to second derivatives. 
5. In the extrapolation polynomial, there is a constraint that the polynomial's value and derivative are equal to the function values at discrete points.  There is a final constraint that an arbitrary higher-order derivative of the polynomial is equal to zero at a specific point yhat ($\hat{y}$).  this value serves as a free parameter for the minimization of the integral for the spectral error.
6. We have extended Kim's method by introducing a new free parameter equivalent to yhat for each boundary row.  So the stencil for each boundary row is optimized.
7. We have created Mathematica Notebooks to perform these calculations.  The file @/Users/dave/projects/HirschmannGroupBYU/Notes/MathematicaNotebooks/CFD\ Interior\ Optimization.nb follows the procedure in Kim07 to produce the interior coefficients for different derivative schemes.  This computes the interior coefficients for both first and second derivatives
8. The file @/Users/dave/projects/HirschmannGroupBYU/Notes/MathematicaNotebooks/CFD\ Boundary\ Optimization.nb follows the methods in Kim24 to generate the boundary coefficients using the boundary closure with the extrapolating polynomial.  This computes the closure for both first and second derivatives
9. The /Users/dave/projects/HirschmannGroupBYU/Notes/MathematicaNotebooks/CFD\ Eigenvalue\ Stability.nb takes finite derivative operators (interior stencils and boundary closures) and computes the eigenvalue spectra for each operator.  It gives a stability determination based on the real part of the eigenvalue.
10. The notebooks can generate C++ code for the operators.
